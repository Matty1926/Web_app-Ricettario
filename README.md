# 🍽️ Il Ricettario di MP

Un'applicazione web raffinata e funzionale per custodire, organizzare e consultare le tue ricette preferite. Progettata con un'estetica classica ispirata ai libri di cucina cartacei, ma con il cuore tecnologico di una moderna **Progressive Web App (PWA)**.

![Versione](https://img.shields.io/badge/versione-2.0-8b3a1a)
![Licenza](https://img.shields.io/badge/licenza-MIT-1a1209)

## ✨ Caratteristiche Principali

- **Design Elegante**: Interfaccia curata con font *Playfair Display* e *Lora* per un'esperienza di lettura superiore.
- **Supporto Offline**: Grazie al Service Worker, l'app funziona perfettamente anche senza connessione internet.
- **Calcolo Dinamico Porzioni**: Regola le quantità degli ingredienti istantaneamente in base al numero di commensali.
- **Modalità Cucina**: Un'interfaccia dedicata per seguire il procedimento passo dopo passo senza distrazioni.
- **Timer Intelligente**: Rilevamento automatico dei tempi di cottura nel testo.
- **Privacy Totale**: I dati sono salvati localmente sul tuo dispositivo; nessuno tranne te ha accesso alle tue ricette.
- **Backup & Ripristino**: Esporta e importa l'intero ricettario in formato JSON con un clic.

## 🚀 Tecnologie Utilizzate

- **HTML5** (Semantico)
- **CSS3** (Variabili, Flexbox, Grid, Mobile-first)
- **JavaScript** (Vanilla JS, LocalStorage, API del Service Worker)
- **Google Fonts** (*Playfair Display* & *Lora*)

## 📲 Installazione su Smartphone

Questa app è una PWA. Per installarla sul tuo telefono:

1. Apri l'URL dell'app in Safari (iOS) o Chrome (Android).
2. Tocca l'icona di **Condivisione** (iOS) o i tre puntini (Android).
3. Seleziona **"Aggiungi alla schermata Home"**.

## 🛠️ Come personalizzare

Se vuoi modificare l'app per te:
1. Clona la repository: `git clone https://github.com/tuo-username/nome-repo.git`
2. Apri `index.html` per modificare la struttura o gli stili.
3. Modifica `manifest.json` per cambiare icone o colori del tema.
